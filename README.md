# Zyro Blog

Welcome to the Zyro blog repository! This site is hosted on GitHub Pages.

## 🌐 Live Site
Visit your blog at: **https://zyrosochlive.qzz.io**

## 📝 How to Update Your Blog

1. Edit the `index.html` file to customize your landing page
2. Add more HTML files or create a blog post structure
3. Commit and push changes to the `main` branch
4. GitHub Pages will automatically deploy your changes

## 🚀 Getting Started

- Modify `index.html` with your content
- Add CSS styling in a `style.css` file
- Create blog posts in separate directories or files
- Update links and navigation as needed

## 📦 Repository Structure


## 🔧 GitHub Pages Configuration

Your site is automatically deployed from the `main` branch. Any changes pushed to this branch will be live within seconds.
